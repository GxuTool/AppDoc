# 开发计划与已知问题

西大工具箱目前仍在积极开发中，部分功能尚未完善，用户体验还有较大的提升空间。 当前版本为 **v2.1.2 beta**，处于公测阶段。大部分核心功能已经可用，但仍有不少体验细节需要打磨。

## 已知的体验问题

### 登录与账号

- **Token 过期无感知**：各系统的登录 Token 过期后，使用功能时才会发现失败，缺少主动检测和提醒机制
- **登录失败提示不够友好**：部分登录失败场景的错误信息不够明确，用户难以自行排查
- **23:00 限制缺少提前提示**：在接近 23:00 时没有提前提醒用户，使用中突然失败体验不好

### 课表与日程

- **课表刷新不够及时**：教务系统上的课程变动不能实时同步到 APP，需要手动刷新
- **冲突课程展示体验待优化**：同一时间段多门课程堆叠显示时，切换查看的操作不够直观
- **课表设置入口较深**：需要点击右上角菜单 → 课表设置，操作步骤较多
- **学期起始日需手动设置**：每学期开学时需要手动调整起始日期，否则周数计算不准确
- **缺少课表导出为图片功能**：当前导出课表是 HTML 格式，不方便直接分享到社交媒体

### 考试与成绩

- **成绩详情加载较慢**：展开平时成绩时需要额外请求，加载过程中显示「正在加载中」等待时间较长
- **缺少成绩趋势分析**：只能查看单次成绩，无法直观看到各学期的绩点变化趋势
- **考试座位号信息**：教务系统隐藏了，考试的座位号，显示为「未知」

### 考勤

- **考勤数据同步延迟**：打卡后需要较长时间才能在 APP 中更新
- **Token 频繁过期**：考勤系统的 Token 过期较快，需要频繁输入验证码重新登录
- **缺少考勤统计**：只能查看逐条记录，缺少出勤率、迟到次数等统计信息

### 通知与消息

- **调课通知解析不够准确**：部分格式不规范的调课通知可能无法正确解析结构化信息
- **缺少消息推送**：新通知（调课、成绩发布等）不能主动推送给用户，需要手动打开查看
- **学校文件功能依赖单独账号**：文件系统的账号与教务系统不同，容易混淆

### 教学评价

- **一键评价后无二次确认**：一键填写后直接进入待提交状态，缺少让用户逐一确认的引导流程
- **模板设置较为简陋**：评价模板的自定义选项有限，不能针对不同类型的课程设置不同模板
- **仅在开放期可用**：非评价开放期间页面显示空白，缺少引导说明

### 界面与交互

- **部分页面加载状态不一致**：有的页面用 ActivityIndicator，有的用文字提示，风格不统一
- **深色模式适配不完整**：少数页面在深色模式下存在文字颜色对比度不足的问题
- **错误处理不够统一**：不同功能的错误提示方式各异，有的用 Toast，有的用横幅，有的用文字
- **返回手势支持不完善**：部分弹窗和底部面板的关闭交互不够顺畅

### 其他功能

- **自主选课功能未完成**：该功能目前被注释掉，尚未对外发布
- **校园网充值功能未完成**：该功能目前被注释掉
- **小部件预览功能未完成**：该功能目前被注释掉
- **选课课程列表缺少分页优化**：一次性加载大量数据，可能导致页面卡顿
