# 功能情景与常见问题

## 账号相关

### Q1：如何保证我登录的校园账号安全？

输入的账号和密码只通过缓存数据的方式，存储在本机 APP 内，不会上传至任何云端服务器。账号信息仅用于完成对应功能的数据查询和展示。

请妥善保管自己的账号和密码，不要将账号信息提供给他人，也不要在不可信设备上保存账号。

### Q2：为什么晚上 11 点后无法使用部分功能？

学校教务系统资源在 23:00 至次日 7:30 仅限校园网网络访问。使用非校园网网络时会被拒绝访问。

系统会提示：「出于网络安全角度考虑，针对校外网络的访问，本系统的开放时间仅为每天的 7:30 - 23:00。其他时间如需访问，请通过广西大学的 VPN 系统进行访问。」

**解决方案：**

- 连接校园网后再使用
- 通过学校 VPN 系统访问
- 在 7:30 后再操作

APP 暂不支持 VPN 连接。

### Q3：为什么有些功能无法正常使用或启用时闪退？

**排查步骤：**

1. **检查账号登录状态**：
   - 首页标题「日程表」左侧的图标颜色可快速判断（绿色 = 已登录，红色 = 未登录）
   - 或进入设置页面查看各账号的详细状态
   - 不同功能需要不同的账号（详见下方对照表）

2. **检查网络连接**：确保网络正常，教务系统 23:00 后限校园网

3. **检查学校系统状态**：学校相关系统可能在维护中

4. **尝试重新登录**：进入设置 → 对应账号设置 → 重新登录

**功能与账号对照表：**

| 功能 | 需要的账号 |
|------|-----------|
| 课表查询、考试信息、成绩查询 | 教务账号 |
| 学校文件查阅 | 统一认证账号 |
| 考勤信息查询 | 考勤系统账号 |

### Q4：三个账号系统有什么区别？

APP 使用三个独立的账号系统：

| 账号系统 | 用途 | 登录入口 |
|----------|------|----------|
| 教务系统账号 | 课表、考试、成绩、选课等核心功能 | 设置 → 教务账号设置 |
| 统一认证系统账号 | 学校文件查阅 | 设置 → 统一认证账号设置 |
| 考勤系统账号 | 考勤信息查询 | 设置 → 考勤系统账号设置 |

这三个账号系统可能使用不同的密码，请根据学校实际情况分别设置。

## 功能使用

### Q5：如何设置个性化主题？

进入设置 → 主题区域：

1. **主题色**：点击颜色选择器，选择喜欢的主题颜色。主题色会影响按钮、导航栏、选中状态等元素的颜色
2. **背景图**：点击「选择图片」从相册选择背景，或点击「重置背景」恢复默认
3. **背景透明度**：拖动滑动条调整背景蒙版的透明程度（0 = 完全遮挡背景，130 = 背景完全清晰）
4. **课程颜色**：点击「清空课程颜色缓存」可重置课表课程块的自定义颜色
5. **课表配色风格**：在首页 → 菜单 → 课表设置中，可选择经典、马卡龙、莫兰迪、鲜明四种配色方案

### Q6：校选课查漏的计算结果是否准确？

校选课查漏功能会根据你的入学年份，自动匹配培养方案要求，并结合已获得的成绩计算。

但此功能的计算结果仅供参考，原因包括：

- 培养方案可能有调整，系统匹配的方案可能不完全准确
- 部分课程的学分认定可能存在特殊情况
- 数据来源于教务系统，可能存在延迟
- 仅计算已及格的课程，挂科课程不计入已获得学分

**实际毕业资格以学校教务处审核为准。建议定期结合系统的分析结果，对照培养方案核实。**

### Q7：考试成绩查询显示「暂无数据」怎么办？

可能的原因：

- 当前学期尚未发布成绩
- 所选学年/学期有误，请核实查询参数
- 教务系统维护中，数据暂时无法获取
- 教务账号登录状态已过期，需要重新登录（进入设置 → 教务账号设置 → 重新登录）

### Q8：考勤信息不准确怎么办？

考勤数据来源于学校考勤系统，可能存在以下情况：

- **数据同步延迟**：打卡后需要一定时间才会在 APP 中更新
- **网络问题**：打卡数据未成功上传到服务器
- **Token 过期**：进入考勤信息页面时如提示 Token 过期，需要输入验证码重新登录

**解决方案：**

1. 下拉刷新考勤数据
2. 检查考勤系统账号登录状态
3. 如有考勤异议，请以学校官方考勤记录为准，并及时联系任课教师或相关管理部门核实

### Q9：如何查看未出分的科目？

进入工具箱 → 信息查询 → 考试成绩，选择对应的学年学期查询后：

- 已出成绩的科目会显示在成绩列表中
- 未出分的科目会单独显示在页面底部的「未出分科目」区域
- 未出分科目以标签按钮形式展示，一目了然

### Q10：调课信息怎么看？

进入工具箱 → 通知 → 调课信息：

1. 顶部切换「未读消息」和「已读消息」查看
2. 开启「仅显示本学期」过滤当前学期的调课通知
3. 点击通知条目展开查看详细变动内容
4. 变动项会以删除线标记原值、绿色粗体标记新值，箭头连接
5. 未变动的字段不显示
6. 展开区域底部显示通知原始全文

### Q11：如何使用导出课表功能？

进入工具箱 → 信息查询 → 导出课表：

1. 选择要导出的课表类型（个人课表/班级课表）
2. 等待页面加载完成（底部按钮变为可点击）
3. 点击「导出课表」按钮
4. 系统获取页面 HTML 内容并保存到本地
5. 自动调用系统打开方式，可以选择用浏览器或其他应用打开

::: tip 提示
导出的是教务系统的原始网页，可以在浏览器中查看或打印。
:::

### Q12：GPA 计算器的计算方法是什么？

进入 GPA 计算器页面，点击「计算方法」旁的问号图标可查看完整公式。

计算依据为《广西大学普通本科学生课程修读、考核及成绩管理办法（2019年修订）》：

- 课程学分绩点 = （成绩 ÷ 满分 × 100）/ 10 - 5
- 不及格课程学分绩点为零
- 平均学分绩点 = Σ（学分绩点 × 已得学分 × K）/ Σ 修读学分

## 其他

### Q13：节假日信息的来源是什么？

- 全国节假日信息来源于国务院办公厅的官方通知
- 广西地方节假日信息来源于广西壮族自治区人民政府办公厅的官方通知

但学校可能会根据实际情况调整（如校历安排），请同时关注学校官方通知。

### Q14：地图导航功能需要什么条件？

使用地图导航功能前，请确保手机上已安装以下地图应用之一：

- 高德地图
- 百度地图

点击导航后会自动跳转到地图应用并输入目的地名称。如未安装地图应用，可能无法正常跳转。

### Q15：一键评价后需要手动提交吗？

是的。一键评价只是自动填写所有课程的评价内容，填写完成后还需要手动点击页面底部的「提交评价」按钮才能最终提交到教务系统。

建议一键评价后逐门检查评价内容，确认无误后再提交。
