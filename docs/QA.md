# 功能情景与常见问题

## Q1：如何保证我登录的校园账号安全？

输入的账号和密码只通过缓存数据的方式，存储在APP内，不会泄露。我们也不会上传到云端服务器。

## Q2：为什么晚上11点后需要使用校园网？

学校资源在23点到隔天6点仅限校园网网络访问，使用非校园网网络会被拒绝访问。

若晚上11点后不使用校园网，将提示“出于网络安全角度考虑，针对校外网络的访问，本系统的开放时间仅为每天的7：30—23：00。其他时间如需访问，请通过广西大学的VPN系统进行访问。”

而APP暂不考虑支持VPN。

## Q3: 为什么有些功能无法正常使用或启用时闪退

- 可能是未登录该功能对应的校园账号导致的，可以在首页左上角观察人头的颜色，对应的账号
若是已登录会显示绿色，或者在设置中检查账号的登录状态。

- 也有可能是该功能在维修中，导致无法正常使用。
