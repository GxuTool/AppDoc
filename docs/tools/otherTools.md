## 学校文件

进入工具箱 → 通知 → 学校文件，可以查看待阅的学校文件。

**使用说明：**

1. 首次使用时需要登录文件系统（文印交换系统）
2. 输入对应的账号和密码进行登录
3. 登录成功后会显示待阅文件列表
4. 点击文件标题可查看文件详情

**功能特点：**

- 支持分页浏览，每页显示 10 条文件
- 文件按时间顺序排列
- 支持翻页查看更多文件

::: tip 提示
学校文件系统的账号密码可能与教务系统不同，请根据实际情况设置。
:::

## 地图导航

进入工具箱 → 其他 → 地图导航，可以查看校内地点并进行导航。

**功能说明：**

- 包含校内绝大部分地点（教学楼、图书馆、食堂、宿舍区等）
- 支持一键跳转手机地图应用进行导航
- 自动输入地点名称，无需手动搜索

::: warning 使用前提
使用地图导航功能前，请确保手机上已安装高德地图或百度地图应用。
:::

## 教学楼平面图

进入工具箱 → 其他 → 教学楼平面图，可以查看常去教学楼的楼层教室分布。

**操作流程：**

1. 选择要查看的教学楼
2. 选择要查看的楼层
3. 查看该楼层的平面图，图中标注了教室门牌号

此功能方便快速找到目标教室的位置，特别适合新生熟悉教学楼布局。

## 全校实时课表

进入工具箱 → 其他 → 全校实时课表，可以查看当前时间段内全校各班级正在上课的课表信息。

可用于了解特定时间段各教学楼的课程安排情况。
