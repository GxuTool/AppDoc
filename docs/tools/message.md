# 信息查询
::: warning 前情提要！！！

信息查询里面的大部分功能都需要登录教务系统账号哦，否则可能无法正常使用
::: 


## 个人课表查询

按序修改查询参数，可以查询指定学期的个人课表，查询结果也以首页课程表相同的形式呈现

## 班级课表查询

按序修改查询参数，可以查询指定班级相应学期的课表。查询结果也以与首页课程表相同的形式呈现。

## 考试考场查询

按序修改查询参数（默认为当前学期），可以查询自己的考试信息。结果简明扼要，并能缓存在手机中，适宜考试前即用即查。

## 考试成绩查询

按序修改查询参数（默认为当前学期），可以查询自己的考试成绩。

考试成绩按发布时间降序排列，最后发布的在最前面。

**考试成绩不足60分的，在查询结果栏呈红色。** 方便点开瞬间定性看挂科情况。

点开成绩栏，可以查看更丰富的内容（平时成绩得分与比例、成绩发布时间、学分和绩点等）。
## 考勤信息查询

将每个需要打卡机打卡的课程按课表的形式分块，绿色代表已打卡成功，红色代表缺勤，黄色代表迟到，蓝色代表未到上课时间，

## 选课课程列表

能够查询校选课的选课情况，例如选课人数，教学班级数以及选修的学分

## 校选课查漏

是一个计算校选课数量是否达标的功能。在这里，会将从教务系统获取的校选课信息聚合整理。<img src="../asset/onlineClass.webp" alt="校选课查漏" width="250">

如图，这个功能会计算每个校选课板块的应得学分与已获得学分，帮助学生查漏校选课

## 教师信息查询

在搜索栏输入教师的 姓名/单位名称/职称/一级学科 ，能够直接获取到符合条件的教师的信息

## 期末学生评价

支持一键评价。（ToDo）


# 通知查询

## 调课信息

发生调动的课程会在这里通知。

## 调休信息

发生调休的日期会在这里通知，调课日期代表的是需要上课的日期，
调休日期代表的是原本的上课时间。

## 法定节假日

在这里能查询到广西大学会放假的节假日，分为全国节假日与广西省内的地方及假日。
