# 考试信息

## 考试考场查询

进入工具箱 → 信息查询 → 考试考场，可以查询考试安排和考场信息。

**操作说明：**

1. 选择查询的学年和学期（默认为当前学期）
2. 点击查询，系统会返回考试信息列表
3. 每页显示 15 条记录，可通过页码切换查看

**考试状态标签：**

查询结果中的每条考试信息会显示状态标签：

| 标签 | 含义 |
|------|------|
| 即将开考 | 考试尚未开始，按时间升序排列 |
| 时间未知 | 考试时间尚未确定 |
| 已结束 | 考试已经结束，按时间降序排列 |

**考试信息卡片包含：**

- 课程名称
- 考试时间
- 考试地点（教室）
- 座位号（未开考的考试才会显示）
- 考试类型

::: tip 提示
查询结果会缓存在手机中，适宜考试前即用即查，无需反复联网。
:::

## 考试成绩查询

进入工具箱 → 信息查询 → 考试成绩，可以查询自己的考试成绩。

**操作说明：**

1. 选择查询的学年和学期（默认为当前学期）
2. 点击查询，系统会返回成绩列表
3. 每页显示 15 条记录，可通过页码切换

**成绩显示规则：**

- 考试成绩按发布时间降序排列，最后发布的在最前面
- **考试成绩不足 60 分的，在查询结果栏呈红色**，方便瞬间定性看挂科情况
- 未出分的科目会单独显示在「未出分科目」区域

**查看成绩详情：**

点击成绩栏可以展开查看更丰富的内容：

- 平时成绩得分与比例
- 成绩发布时间
- 学分
- 绩点
- 教师姓名

**绩点计算器：**

查询页面底部有「绩点计算器」按钮，点击可进入 GPA 计算器，手动输入各科成绩和学分进行加权平均绩点计算。

::: warning 重要提醒
考试成绩以广西大学教务系统显示为准。如对成绩有疑问，请及时联系任课教师或教务处。
:::
