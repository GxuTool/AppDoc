# 教学评价

## 期末学生评价

进入工具箱 → 教学评价 → 期末学生评价，可以在期末教学评价开放期间完成学生评价。

::: warning 使用前提
此功能仅在教务系统开放学生评价期间才可正常使用。非开放时间会显示「暂无待评价课程」提示，说明当前可能未到期末学生评价开放时间，或所有课程暂未发布评价任务。
:::

### 页面布局

**顶部信息卡片：**

页面顶部显示统计信息卡片，包含：
- 待评价课程总数
- 各状态的课程数量统计

**操作按钮区域：**

信息卡片下方有两个主要操作按钮：

| 按钮 | 说明 |
|------|------|
| 一键评价 | 使用预设模板自动填写所有课程的评价 |
| 模板 | 进入评价模板设置页面，自定义一键评价使用的模板 |

**评价列表：**

以表格形式展示所有待评价课程，每行显示：

| 列名 | 说明 |
|------|------|
| 课程 | 课程名称 |
| 教师 | 授课教师姓名 |
| 状态 | 评价状态（带颜色标识） |

**评价状态说明：**

| 状态 | 颜色 | 含义 |
|------|------|------|
| 已评完 | 绿色 | 该课程的评价已完成 |
| 未评完 | 黄色 | 该课程的评价部分完成 |
| 未评 | 红色 | 该课程尚未开始评价 |

点击任意课程行可以进入该课程的评价详情页面。

### 一键评价

一键评价功能可以快速完成所有课程的评价，使用预先设置的评价模板自动填写。

**操作流程：**

1. 点击「一键评价」按钮
2. 弹出进度对话框，显示「正在一键评教...」
3. 进度条实时显示评价进度
4. 评价过程中可以点击「取消」按钮中止操作
5. 所有课程评价完成后，进度显示「完成」

**注意事项：**

- 一键评价使用的是评价模板中设置的内容，所有课程使用相同的评价
- 一键评价只是填写评价内容，不会自动提交，还需要手动点击「提交评价」按钮

### 评价模板

点击「模板」按钮进入评价模板设置页面，可以自定义一键评价使用的评价内容。

**模板设置：**

- 可以设置各个评价维度的评分和评语
- 保存后，将使用此模板自动一键评价
- 建议根据实际情况设置合理的评价内容

### 逐门评价

如需个性化评价特定课程（如某门课程特别好或特别差），可以逐门评价：

1. 在课程列表中点击要评价的课程
2. 进入评价详情页面
3. 按要求填写各项评价内容（包括评分选项和文字评语）
4. 保存评价

### 提交评价

所有课程评价完成后（状态全部变为「已评完」），页面底部会出现「提交评价」按钮。

点击「提交评价」将所有评价一次性提交到教务系统。

::: warning 重要提醒
- 提交评价前请仔细确认所有课程的评价内容，即使提交后可以取消评价
- 取消评价是未列于教务系统官网的实验性功能，我们无法保证该操作的效果和影响
- 使用一键评价后，建议逐门核对确认，避免因模板内容不适合某些课程而产生不准确的评价
- 评价是教学反馈的重要渠道，建议认真对待，提供真实客观的评价
:::
