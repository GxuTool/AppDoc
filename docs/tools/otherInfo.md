# 其他信息

## 考勤信息查询

进入工具箱 → 信息查询 → 考勤信息，可以查看需要打卡机打卡的课程的出勤状态。

**功能说明：**

将每个需要打卡的课程按课表的形式分块显示，颜色标识含义如下：

| 颜色 | 含义 |
|------|------|
| 🟢 绿色 | 已打卡成功 |
| 🔴 红色 | 缺勤 |
| 🟡 黄色 | 迟到 |
| 🔵 蓝色 | 未到上课时间 |

::: tip 提示
使用此功能前需要先在设置中配置考勤系统账号。
:::

::: warning 注意
考勤数据来源于学校考勤系统，可能存在延迟。如有考勤异议，请以学校官方考勤记录为准，并及时联系任课教师核实。
:::

## 教师信息查询

进入工具箱 → 信息查询 → 教师信息查询，可以搜索查询教师的相关信息。

**搜索方式：**

在搜索栏输入以下任一维度的关键字即可搜索：

- 教师姓名
- 单位名称（如学院名称）
- 职称（如教授、副教授）
- 一级学科

搜索结果会显示符合条件的教师信息列表，方便了解教师基本情况。
