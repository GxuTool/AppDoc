# 课程信息

## 个人课表查询

按序修改查询参数，可以查询指定学期的个人课表，查询结果也以首页课程表相同的形式呈现

## 班级课表查询

按序修改查询参数，可以查询指定班级相应学期的课表。查询结果也以与首页课程表相同的形式呈现。

## 物理实验课

在这里能够直接获取到物理实验课的详细信息，包括上课地点，任课教师，实验名称等内容

## 金工实训

在这里能够获取到金工实训的信息，免于登录工程训练中心

## 选课课程列表

能够查询校选课的选课情况，例如选课人数，教学班级数以及选修的学分

## 校选课查漏

是一个计算校选课数量是否达标的功能。在这里，会将从教务系统获取的校选课信息聚合整理。
<img src="../assets/onlineClass.webp" alt="校选课查漏" width="250">

如图，这个功能会计算每个校选课板块的应得学分与已获得学分，帮助学生查漏校选课

## 全校实时课表

能够获取到当前时间段内的全校课表
