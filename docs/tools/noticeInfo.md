# 通知信息

## 调课信息

进入工具箱 → 通知 → 调课信息，可以查看教务系统发布的课程调动通知。

**操作说明：**

- 页面顶部可以切换「未读消息」和「已读消息」
- 开启「仅显示本学期」开关，可以过滤出当前学期的调课通知
- 点击通知条目可以展开查看详细的调课内容

**调课详情包含：**

- 课程名称
- 教师变动（原教师 → 新教师）
- 周次变动
- 星期变动
- 节次变动
- 地点变动
- 原始通知全文

变动项会以 ~~删除线~~ 标记原值、绿色粗体标记新值，方便快速识别变化内容。

::: warning 注意
发生调动的课程请及时关注，以免走错教室或错过课程。建议在上课前再次确认课程安排。
:::

## 调休信息

进入工具箱 → 通知 → 调休信息，可以查看调休日期的通知。

**基本概念：**

- **调课日期**：代表需要上课的日期（原本不上课的日期变为上课日）
- **调休日期**：代表原本的上课时间变为休息日

::: tip 提示
调休信息涉及课程安排的临时调整，请及时关注并做好相应安排。
:::

## 法定节假日

进入工具箱 → 通知 → 法定节假日，可以查看广西大学放假的节假日安排。

**功能说明：**

页面顶部有两个标签页可供切换：

- **全国节假日**：显示国务院发布的全国性节假日安排，包含元旦、春节、清明节、劳动节、端午节、中秋节、国庆节
- **地方节假日**：显示广西壮族自治区的地方节假日安排（如三月三）

**节假日卡片包含：**

- 节假日名称
- 放假时间范围
- 放假天数
- 调休补班安排（如有）

补班信息会以红色文字提醒，便于提前规划。

::: warning 注意
节假日安排以国务院办公厅和广西壮族自治区人民政府办公厅的官方通知为准。学校可能会根据实际情况进行调整，请同时关注学校官方通知。
:::
