# 通知信息

## 调课信息

进入工具箱 → 通知 → 调课信息，可以查看教务系统发布的课程调动通知。

**页面布局：**

页面顶部有消息类型切换按钮组，包含两个选项：

| 按钮 | 说明 |
|------|------|
| 未读消息 | 显示尚未阅读的调课通知（默认选中） |
| 已读消息 | 显示已经阅读过的调课通知 |

切换按钮后会自动刷新对应类型的列表。

**过滤功能：**

消息类型下方有「仅显示本学期」开关：
- 开启后只显示当前学期的调课通知
- 关闭后显示所有学期的调课通知

**通知列表：**

每条通知以可折叠的列表项显示：

- **标题**：显示「调课提醒：课程名称」
- **时间**：通知的发布时间（灰色小字）

点击通知条目可以展开/收起详细内容。

**调课详情内容：**

展开后显示详细的调课信息，自动解析通知文本并结构化展示变动内容：

| 字段 | 说明 |
|------|------|
| 教师 | 教师变动（如有） |
| 周次 | 上课周次变动（如有） |
| 星期 | 上课星期变动（如有） |
| 节次 | 上课节次变动（如有） |
| 地点 | 上课地点变动（如有） |

**变动标识规则：**

- 未变动的字段不显示
- 原值以 ~~删除线~~ 标记，颜色为灰色
- 新值以 **绿色粗体** 标记
- 箭头图标（→）连接原值和新值

展开区域底部还会显示通知的原始全文，方便查看完整的调课说明。

::: warning 注意
发生调动的课程请及时关注，以免走错教室或错过课程。建议在上课前再次确认课程安排是否正确。
:::

## 调休信息

进入工具箱 → 通知 → 调休信息，可以查看学校发布的调休日期通知。

**页面内容：**

以表格形式展示调休信息，每行包含两列：

| 列名 | 说明 |
|------|------|
| 调课日期 | 需要上课的日期（原本不上课的日期变为上课日） |
| 调休日期 | 原本的上课时间变为休息日 |

日期格式为「YYYY年MM月DD日」，调休日期还会额外显示星期几。

::: tip 提示
调休信息与调课信息不同。调课是课程时间/地点的变动，调休是整个日期的上班/休息调整。两者都会影响课程安排，请都及时关注。
:::

## 法定节假日

进入工具箱 → 通知 → 法定节假日，可以查看广西大学放假的节假日安排。

**页面布局：**

页面顶部有两个标签页可供切换：

| 标签 | 说明 |
|------|------|
| 全国节假日 | 显示国务院办公厅发布的全国性节假日安排 |
| 地方节假日 | 显示广西壮族自治区人民政府办公厅发布的地方节假日安排（如三月三） |

**全国节假日页面：**

顶部显示通知标题和文号（如「国办发明电〔2025〕7号」），以及通知正文的开头段落。

每个节假日以卡片形式显示，包含：

- **节假日名称**：如「元旦」「春节」等，以主题色显示
- **放假天数**：右上角以标签形式显示（如「共9天」）
- **放假时间范围**：具体的放假日期
- **调休补班安排**：如有补班，以红色文字（⚠️ 调休：...）显示补班日期

底部显示通知的结尾段落和发文单位、日期。

**地方节假日页面：**

广西地方节假日主要显示「三月三」的放假安排，结构与全国节假日类似。

::: warning 注意
节假日安排以国务院办公厅和广西壮族自治区人民政府办公厅的官方通知为准。学校可能会根据实际情况调整，请同时关注学校官方通知。
:::
