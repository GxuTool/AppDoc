# 声明

## 非官方说明

西大工具箱是面向广西大学学生的校园辅助工具，旨在提升课表、考试、成绩、考勤等信息的查询体验。

本软件并非广西大学官方应用，文档内容仅用于说明软件功能和使用方式。

## 数据来源

软件中展示的课表、考试、成绩、考勤等信息主要来源于学校的相关系统。

由于学校系统可能存在维护、网络限制、接口调整、数据延迟等情况，软件展示的信息可能与学校官方系统存在差异。

涉及考试、选课、成绩、考勤等重要事项时，请以学校官方系统和学院通知为准。

## 账号与隐私

输入的账号和密码只通过缓存数据的方式保存在本机，存储在APP内用于访问学校相关系统，不会泄露。我们也不会上传到云端服务器。

账号信息仅用于完成对应功能的数据查询和展示。请用户妥善保管自己的账号和密码，不要将账号信息提供给他人，也不要在不可信设备上保存账号。


## 问题反馈

如果你在使用过程中发现功能异常、数据错误或文档描述不准确，可以通过公测群联系开发人员反馈问题。
