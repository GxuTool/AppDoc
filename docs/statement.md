# 声明

## 非官方说明

西大工具箱是面向广西大学学生的校园辅助工具，旨在提升课表、考试、成绩、考勤等信息的查询体验。

**本软件并非广西大学官方应用**，与广西大学无任何隶属或合作关系，不代表广西大学立场，不受其指示，也未获得来自广西大学的直接支持。本项目中的所有功能实现及界面重构均由开发者基于公开、合法的技术手段独立完成。

文档内容仅用于说明软件功能和使用方式。

## 数据来源

软件中展示的课表、考试、成绩、考勤等信息主要来源于广西大学的相关系统（教务系统、考勤系统等）。

由于学校系统可能存在维护、网络限制、接口调整、数据延迟等情况，软件展示的信息可能与学校官方系统存在差异。

**涉及以下重要事项时，请务必以学校官方系统和学院通知为准：**

- 考试安排确认
- 成绩复核
- 选课操作
- 考勤核对
- 毕业资格审查
- 其他涉及学业的重要决策

## 工具辅助性质

本软件提供的各项功能仅作为辅助工具使用，计算结果和展示信息仅供参考：

| 功能      | 说明                                    |
|---------|---------------------------------------|
| 校选课查漏   | 基于系统数据与公开文件自动推算，匹配培养方案可能存在偏差，计算结果仅供参考 |
| GPA 计算器 | 基于教务系统返回的成绩数据计算，实际绩点以教务系统为准           |
| 一键评价    | 仅简化操作流程，用户应对评价内容负最终责任，建议逐门核对后提交       |
| 节假日信息   | 来源于官方通知，但学校可能有特殊安排，以学校通知为准            |
| 考勤信息    | 数据来源于考勤系统，可能存在同步延迟，以学校官方记录为准          |
| 教师信息    | 数据来源于学校公开信息，如有变动以学校最新信息为准             |

**用户在面临重要决策时，应自行核实相关信息，保持独立判断。不应完全依赖本软件提供的辅助功能结果。**

## 账号与隐私

- 输入的账号和密码只通过缓存数据的方式保存在本机，存储在 APP 内用于访问学校相关系统
- 账号密码不会上传至任何云端服务器
- 账号信息仅用于完成对应功能的数据查询和展示
- APP 不收集用户的个人信息用于其他目的

请用户妥善保管自己的账号和密码，不要将账号信息提供给他人，也不要在不可信设备上保存账号。
