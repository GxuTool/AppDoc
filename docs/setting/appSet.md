# 软件设置

进入底部标签栏 → 设置，可以管理账号、个性化主题以及查看软件信息。

## 账号设置

### 查看个人信息

点击「查看个人信息」可以查看在教务系统中登记的个人基本信息，包括学号、姓名、学院、专业等。

::: tip 提示
需要先登录教务账号才能查看个人信息。如显示「暂无个人信息」，请检查教务系统的登录状态。
:::

### 教务账号设置

教务账号是使用大部分核心功能的基础。点击「教务账号设置」进入登录页面：

1. 输入在广西大学教务系统上注册的账号和密码
2. 点击登录按钮
3. 登录成功后会显示绿色状态标识

::: warning 注意事项
- 账号密码仅保存在本机，不会上传至云端
- 学校教务系统在每日 23:00 至次日 6:00 仅限校园网访问
- 如登录失败，请检查账号密码是否正确，或确认是否需要通过 VPN 访问
:::

### 统一认证系统账号设置

部分功能（如学校文件查阅）需要统一认证系统的账号。点击「统一认证系统账号设置」进行配置。

### 考勤系统账号设置

如需使用考勤信息查询功能，需要设置考勤系统账号。点击「考勤系统账号设置」进行配置。

## 主题设置

在设置页面的主题区域，可以自定义应用的外观风格。

### 主题色

点击主题色区域的颜色选择器，可以自定义应用的主题颜色。

主题色会影响：
- 按钮颜色
- 导航栏高亮
- 选中状态颜色
- 其他强调色元素

### 课程颜色

点击「清空课程颜色缓存」按钮，可以重置课表中各课程的自定义颜色，恢复默认配色。

### 背景图

- **选择图片**：从手机相册中选择一张图片作为应用背景
- **重置背景**：清除自定义背景图，恢复默认背景

### 背景蒙版相对透明度

通过滑动条调节背景蒙版的透明度：

- 数值越大，背景越清晰（蒙版越透明）
- 数值越小，背景越模糊（蒙版越不透明）
- 范围为 0 - 130

根据所选背景图的明暗程度，适当调节透明度可以达到最佳视觉效果。

## 软件信息

### 意见反馈

点击「意见反馈」可以向开发团队提交使用中遇到的问题或建议。

填写反馈内容时请尽量详细描述问题，包括：
- 问题出现的操作步骤
- 问题的具体表现
- 手机型号和系统版本（如有必要）

### 开发模式

开发模式是面向开发者和高级用户的调试功能，开启后会在部分页面显示额外的调试信息（如原始 API 数据）。

普通用户无需开启此功能。

### 代码版本号

显示当前安装的应用版本号，在反馈问题时建议提供此信息。

### 源代码

点击可查看应用的 GitLab 源代码仓库。

### 软件文档

点击可跳转至本文档站点。

### 公测群

点击可加入西大工具箱公测 QQ 群（群号：580048805），获取最新动态和参与讨论。

### 软件信息

显示应用的版权信息和开发者联系方式。
