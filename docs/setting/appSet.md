# 软件设置

进入底部标签栏「设置」标签页，可以管理账号、个性化主题以及查看软件信息。

## 账号设置

### 查看个人信息

点击「查看个人信息」可以查看在教务系统中登记的个人基本信息。

**显示内容：**

- 学号
- 姓名
- 学院
- 专业
- 年级
- 班级
- 其他教务系统中的个人信息

信息以列表形式逐行显示，左侧为字段名称（灰色），右侧为对应的值（黑色加粗）。

**加载状态：**

- 加载中：显示加载动画和「加载中....」提示
- 无数据：显示「暂无个人信息，请检查教务系统登录状态」
- 加载完成：显示所有个人信息列表

::: tip 提示
需要先登录教务账号才能查看个人信息。如显示「暂无个人信息」，请先到「教务账号设置」完成登录。
:::

### 教务账号设置

教务账号是使用大部分核心功能的基础。详见 [基础教程 - 教务账号设置](../guide/main.md#教务账号设置)。

**登录页面元素：**

- **状态指示器**：页面顶部显示当前登录状态（灰色圆点 = 未配置，黄色 = 已保存未登录，绿色 = 已登录）
- **状态文字**：显示具体状态说明
- **加载动画**：正在操作时显示
- **结果横幅**：登录成功（绿色边框）或失败（红色边框）时显示，包含标题和详细信息
- **学号输入框**：输入教务系统学号，左侧有用户图标
- **密码输入框**：输入教务系统密码，左侧有锁图标，右侧有眼睛图标可切换密码显示/隐藏
- **登录按钮**：点击执行登录操作
- **打开教务登录页按钮**：在内置浏览器中打开教务系统登录页面
- **提示文字**：「仅供工具从教务系统获取信息」「23:00 至次日 7:30 请连接校园网」

### 统一认证系统账号设置

部分功能（如学校文件查阅）需要统一认证系统的账号。操作方式与教务账号类似。

### 考勤系统账号设置

如需使用考勤信息查询功能，需要设置考勤系统账号。操作方式与教务账号类似。

::: warning 注意事项
- 所有账号密码仅保存在本机，不会上传至任何云端服务器
- 请妥善保管自己的账号信息，不要将账号提供给他人
- 学校教务系统在每日 23:00 至次日 7:30 仅限校园网访问
:::

## 主题设置

在设置页面的主题区域，可以自定义应用的外观风格。

### 主题色

点击主题色区域的颜色选择器，可以自定义应用的主题颜色。

**影响范围：**

主题色会影响以下元素的颜色：
- 按钮颜色
- 导航栏高亮色
- 选中状态颜色
- 卡片边框和标题色
- 其他强调色元素

操作方式：点击颜色区域，弹出颜色选择器，选择喜欢的颜色后自动应用。

### 课程颜色

点击「清空课程颜色缓存」按钮，可以重置课表中各课程的自定义颜色。

清除后，课表中的课程块会恢复为默认的配色方案。如果你之前为不同课程设置了不同的颜色，清除后需要重新设置。

### 背景图

- **选择图片**：点击「选择图片」按钮，从手机相册中选择一张图片作为应用的背景图
- **重置背景**：点击「重置背景」按钮，清除自定义背景图，恢复默认背景

选择图片后，应用的各个页面背景会显示该图片，并叠加蒙版以保证内容可读性。

### 背景蒙版相对透明度

通过滑动条调节背景蒙版的透明度（范围 0 - 130）：

| 数值 | 效果 |
|------|------|
| 0 | 蒙版完全不透明，背景图完全被遮挡 |
| 65 | 蒙版半透明，背景图隐约可见（默认值附近） |
| 130 | 蒙版几乎透明，背景图清晰可见 |

根据所选背景图的明暗程度，适当调节透明度可以达到最佳视觉效果。背景图较暗时可以降低透明度，背景图较亮时可以提高透明度。

## 软件功能

### 意见反馈

点击「意见反馈」可以进入反馈页面，向开发团队提交使用中遇到的问题或建议。

**反馈页面包含：**

- 反馈内容输入区域
- 提交按钮

**填写建议：**

提交反馈时请尽量详细描述：
- 问题出现的操作步骤
- 问题的具体表现
- 期望的正确行为
- 手机型号和系统版本（如有必要）

### 开发模式

开发模式是面向开发者和高级用户的调试功能。

**开启后的效果：**

- 首页日程表底部会显示「查看课程数据」「查看实践课数据」等调试按钮
- 考试信息页面底部会显示「查看考试考场数据」按钮
- 考试成绩页面底部会显示「查看考试成绩数据」「查看未出分科目」按钮
- 考勤信息页面底部会显示调试数据按钮

点击调试按钮可以查看对应的原始 API 返回数据（JSON 格式），方便排查问题。

::: tip 提示
普通用户无需开启此功能。调试数据仅供开发人员排查问题使用。
:::
