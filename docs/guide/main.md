# 基础教程

## 登录

首次使用时，需要设置相关账号才能正常使用各项功能。

打开 APP → 底部标签栏点击「设置」→ 进入账号设置区域，可以看到以下账号设置入口：

### 教务账号设置

教务账号是使用大部分核心功能（课表查询、考试信息、成绩查询等）的前提。

1. 点击「教务账号设置」
2. 输入在广西大学教务系统上注册的账号和密码
3. 点击获取，系统会尝试登录教务系统

::: tip 提示
登录成功后，首页左上角的人头图标会显示绿色，表示教务账号已正常连接。
:::

### 统一认证账号设置

部分功能（如学校文件查阅）需要统一认证系统的账号。

1. 点击「统一认证账号设置」
2. 输入统一认证系统的账号和密码

### 考勤系统账号设置

如需使用考勤信息查询功能，需要设置考勤系统账号。

1. 点击「考勤系统账号设置」
2. 输入考勤系统的账号和密码

::: warning 注意
输入的账号和密码仅通过缓存数据的方式保存在本机，不会上传至任何云端服务器。请妥善保管自己的账号信息。
:::

## 首页

登录教务系统账号成功后，首页会自动获取并展示个人课表信息。

### 日程表

日程表是首页的核心功能，自动适配当前周目和对应的课程信息。

**基本操作：**

- **查看课程详情**：点击课表上的课程，可以查看简明扼要的课程信息（上课教室、教师、班级 QQ 群号等）
- **手动刷新**：点击右上角刷新按钮，可以从教务系统手动更新课表
- **课表设置**：点击设置按钮，可以自定义课表风格（主题色、课程颜色等）
- **调节高度**：调节课表高度至适宜比例，可以一屏展示全天课表

**特殊标记：**

- 课表上带有 **「调」** 字标记的课程，表示该课程发生过调课，应注意调课的时间和地点
- 当教务系统上有考试信息时，考试信息会自动按照其安排填充到当天课表上，并冠以 **「考试」** 的提示字样

### 更新通知

首页顶部会显示应用更新通知卡片，当有新版本发布时会提示用户更新。

## 底部标签栏

应用底部有三个主要标签页：

- **首页**：查看课表和日程安排
- **工具箱**：各类教务信息查询工具的集合
- **设置**：账号管理、主题个性化、软件信息等

每个标签页顶部都有「打开教务」按钮，点击可快速跳转至教务系统网页版。

## 个性化设置

在「设置」页面可以自定义应用的外观：

- **主题色**：选择应用的主题颜色
- **课程颜色**：自定义课表中各课程的显示颜色
- **背景图**：设置自定义背景图片
- **背景蒙版透明度**：调节背景蒙版的透明程度
